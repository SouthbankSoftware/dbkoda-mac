## Requirement
* Node 8.9.1
* Yarn
* node-gyp: `yarn global add node-gyp` or `npm install -g node-gyp`
* JDK 1.8.0
* **_optional_** Python 2.7 (for testing)

## Build from source
1. `yarn install`
2. `yarn gulp`
